# OpenMail Agent Skills

Agent skills for [OpenMail](https://openmail.sh) — give AI agents a real email address.

## Install

```bash
npx skills add openmailsh/skills
```

## Skills

| Skill | Description |
|---|---|
| [openmail](./skills/openmail/SKILL.md) | Dedicated email address for sending, receiving, and replying. |
